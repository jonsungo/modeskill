# Changelog

## v0.1

- Established `modeskill` as Modelab's unified private Skill with `ui-consistency` as the only implemented module.
- Added shared Workspace, authorization, evidence, path security, discovery, and local storage models.
- Added multi-project discovery for monorepository-style roots, manual markerless projects, replaceable discovery cache, and stable relative role selections.
- Added a loopback-only, standard-library Workspace Configurator.
- Added structured Draft 2020-12 Schemas for Workspace, Project Profile, and consistency reports.
- Added HTML and JavaScript examples for form semantics, dialog dismissal, and focus restoration.
- Added negative tests for path escape, symlinks, discovery depth, reference immutability, target authorization, report counts, and local storage.
- Documented that v0.1 is not a complete automatic scanner and does not automatically modify real projects.
- Added Chinese/English Configurator localization with browser language persistence.
- Added fixed Git-synced and local-only module distribution, secure repository-root resolution, and local module enablement.
- Added local backup guidance and path-copy support without Git automation or distribution migration.
- Hardened local module and Workspace storage against file, directory, and symbolic-link escapes.
- Added local HTTP Host, Origin, JSON, size, method, and strict-type validation.
- Added bilingual keyboard-accessible explanations for technical Configurator terms.
